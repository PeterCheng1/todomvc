# vue-todos
> vue-todos是一个界面简约，功能丰富的todoList-app.包含响应式设计，pc端，移动端。此次项目的界面配色设计和具体功能是参照[@liangxiaojuan]()的vue-todos，真的非常之感谢。

# 技术架构 
> vue2.0 + vue-router + axios + mock.js + less

# 运行

```
# 更新依赖包
npm install
# 本地运行 at localhost:8080
npm run dev
# 打包
npm run build

```
